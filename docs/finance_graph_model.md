# Finance Graph Model

The graph layer models typed relations such as same sector, same industry, supplier-customer, competitor, parent-subsidiary, lender-borrower, index member, ETF holding, commodity exposure, currency exposure, rates exposure, options underlying, futures underlying, and macro sensitivity.

Feature extraction includes PageRank, degree, weighted degree, relation-specific exposure propagation, corrected-correlation graph inputs, multifractal similarity, and fundamental similarity. The modeling boundary is signal evidence and diagnostics only; Sourceflow does not make truth claims or investment advice.

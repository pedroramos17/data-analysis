# Next-Generation Finance Roadmap

The first layer is deterministic ingestion, normalization, feature flags, multifractal features, corrected correlations, graph features, fundamental features, and leakage-controlled dataset manifests.

The next modeling layer is specified but disabled by default: Multi-head Cross-Attention + Improved GRU + GNN. It combines temporal market and multifractal features, fundamentals available at event time, relation and correlation graphs, and latent market state vectors. Evaluation should use walk-forward, purged, and embargoed validation with IC, rank IC, AUC, balanced accuracy, MSE, Sharpe, Sortino, Calmar, Melao-inspired score, max drawdown, and turnover.

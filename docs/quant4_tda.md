# Quant4 TDA

Quant4 TDA utilities are local-first topology diagnostics for graph and regime
research. The MVP uses dependency-free fallbacks:

- degree-entropy graph complexity
- graph density
- recurrence matrices and recurrence rate
- density change-point detection
- TDA-complexity graph builder based on path variation

Heavy topology dependencies remain optional. Missing PMFG, TMFG, random-matrix,
or advanced TDA libraries must fail with clear optional-dependency messages.

These tools are validation and exploration aids. They do not claim causal
discovery, factor validity, or expected profitability.
